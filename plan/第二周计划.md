### Socket格式

*客户端发向服务端套接字格式：*

| 操作码 1byte | 发送源 8bytes |   内容   |      备注      |
| :----------: | :-----------: | :------: | :------------: |
|      0       |    用户名     |   null   | 登陆注册用户名 |
|      1       |    用户名     | 聊天信息 |  发送聊天内容  |
|              |               |          |                |
|              |               |          |                |
|              |               |          |                |
|              |               |          |                |
|              |               |          |                |
|      8       |    用户名     |   null   |   退出聊天室   |
|              |               |          |                |

*服务端发向客户端套接字格式：*

| 操作码 1byte |  发送源 8bytes  | 内容 |        备注        |
| :----------: | :-------------: | :--: | :----------------: |
|      0       |    登陆用户     | null | 确认登陆，广播通知 |
|      1       | 发送用户/Server | 信息 | 发送聊天/通知内容  |
|              |                 |      |                    |
|              |                 |      |                    |
|              |                 |      |                    |
|              |                 |      |                    |
|              |                 |      |                    |
|      8       |    退出用户     | null |      广播通知      |
|      F       |     服务器      |      |     聊天室满员     |
|      E       |     服务器      |      |  当前用户名已存在  |
